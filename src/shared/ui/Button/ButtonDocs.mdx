# Button

> Компонент кнопки
<br/>

## Props

**`className`** `[string]?` css классы

**`theme`** `см. примеры Themes` темы кнопок

**`mode`** `['box' | 'rounded']?` внешний вид ссылок

**`Icon`** `[FC<IconProps>]?` svg иконка в jsx

**`loading`** `[boolean]?` доп. свойство, которое делает кнопку не активной. Использовать при асинхронных операциях (например - отправка формы)
<br/>

> Также компонент принимает стандартные HTML аттрибуты для тега button **`disabled`**, **`type`** и прочие
<br/>
