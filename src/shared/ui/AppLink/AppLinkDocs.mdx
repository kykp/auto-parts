# AppLink

> Компонент ссылки для всего проекта
<br/>

## Props

**`className`** `[string]?` css классы

**`to`** `[string]` путь перехода

**`isNative`** `[boolean]?` если нужно использовать стандартый a href="" заместо Link to=""

**`theme`** `['default' | 'main' | 'clear']?` внешний вид ссылок

> Также компонент принимает стандартные HTML аттрибуты для тега a

**`target='_blank'`** **`rel='noreferrer'`** и прочие
