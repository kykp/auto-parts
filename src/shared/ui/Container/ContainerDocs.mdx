# Container

> Контейнер для блоков, который включает в себя стандартные отступы, а также имеет определенные размеры для блоков
<br/>

## Props

**`className`** `[string]?` css классы

**`size`** `['xs' | 'm' | 'l']?` возможные размеры контейнера ( xs - 468, m - 990, l - 1440 )
